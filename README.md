# Jejak Jamu 🌿

Jejak Jamu is a mobile application that uses an AI-powered camera to identify traditional herbs and roots, providing users with their health benefits and recipes.

🎥 **Live Demo: https://youtu.be/ypMQfGmBc7o?si=fEw3IbO9kxXLAnoo**

---

## Features

* **AI Camera Detection** — Scan raw ingredients to identify them instantly.
* **Health Benefits** — Access information on the medicinal properties of detected herbs.
* **Recipe Library** — View step-by-step instructions for brewing traditional jamu.

---

## Built With

* React Native
* Expo (Expo Camera)

## Getting Started

Follow these steps to set up and run the project locally.

### Prerequisites

Ensure you have Node.js and the Expo Go app (or Expo CLI) installed on your machine.

* Node.js (v18 or higher recommended)
* npm or yarn
* Expo Go app on your mobile device (for testing)

### Installation

1. Clone the repo
```bash
   git clone https://github.com/your-username/jejak-jamu.git
```
2. Navigate into the project directory
```bash
   cd jejak-jamu
```
3. Install dependencies
```bash
   npm install
```
4. Start the Expo development server
```bash
   npx expo start
```
5. Scan the QR code with Expo Go on your mobile device

---

## Usage

Open the app, point your camera at a raw herb or root, and let the AI model identify it. Tap the result to view its health benefits and jamu recipes.
